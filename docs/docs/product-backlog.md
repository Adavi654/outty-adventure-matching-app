# Initial Product Backlog (12 User Stories)

## PBI-1 User Registration 
As a new user, I want to register an account so I can use the platform.
Acceptance Criteria 
- Email verification
- Password hashing
- User saved to database

## PBI-2 User Login 
As a registered user, I want to be able to log in to my account so that I can access my profile securely

## PBI-3 Profile creation
As a user, I want to create a profile of my adventures so that others know more about me.

## PBI-4 Update Profile
As a user, I want to edit my profile so that my data is up to date.

## PBI-5 Choose Adventure Interests
As a user I want to choose adventure activities so that I can be matched with like-minded people.

## PBI-6 Skill Level 
As a user, I want to set my level of experience, so that I can find appropriate partners.

## PBI-7 Upload Profile Photos
As a user I want to upload my photos to my profile so that other people can see who I am. 

## PBI-8 Identify Potential Matches 
As a user I want to see possible adventure partners so that I can find matches that would be compatible

## PBI-9 Swipe Right/Swipe Left 
As a user, I want to like or pass on profiles so I can show my interest.

## PBI-10 Create Mutual Match 
As a user I want to get notified of mutual matches so that I can interact with users that are interested

## PBI-11 Send Messages 
As a matched user, I want to be able to message so that I can plan adventures.

## PBI-12 Connect Social Media 
As a user I want to link my Instagram account so that others can know more about me.
