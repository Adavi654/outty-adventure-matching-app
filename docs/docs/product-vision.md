## Product Vision Statement

### Far Vision

Outta will become the leading adventure-focused social platform which will allow outdoor lovers
to make important connections based on shared interests, compatible skill levels, and passion
for exploration. Unlike traditional dating application, Outty will focus on building genuine
relationships through outdoor experiences and adventures.

### Near Vision

Develop a Minimum Viable Product (MVP) that allows user to:
- Register and lon into the platform
- Create and manage adventure profiles
- Select adventure interests and skill levels
- Discover nearby users with similar interests
- Swipe right or left on possible adventure partners
- Generate matches when interest is mutual
- Exchange messages with matched users
